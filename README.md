# Python豆瓣图书推荐系统 python172

## 项目概述


## 技术栈

- 后端：Django
- 数据库：MySQL
- 前端：Vue

## 功能模块

- **用户模块**
  - 登录注册
  - 个人中心
- **图书信息模块**
  - 数据爬取
  - 小说信息管理
  - 图书借阅管理
- **数据分析**
  - 数据分析可视化大屏
- **系统管理**
  - 用户管理
  - 图书资讯管理
  - 系统简介

## 系统角色

- 普通用户：登录、注册、查看图书信息、借阅图书等
- 管理员：管理用户、图书信息、系统资讯等

## 运行环境

- Python 3.x
- Django 2.x
- MySQL 5.x
- Vue.js 2.x

## 部署步骤

1. 准备Python运行环境
2. 安装依赖库：根据`requirements.txt`文件进行安装
3. 配置数据库：根据项目需求配置MySQL数据库
4. 运行后端服务：启动Django项目
5. 运行前端服务：启动Vue项目
